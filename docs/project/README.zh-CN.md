## 范围

项目快照是对开放PR、问题和文档健康状况的有时限评估。使用这些来：

- 识别由功能工作驱动的文档缺口
- 在代码更改的同时优先考虑文档维护
- 跟踪随时间演变的PR/问题压力

对于稳定的文档分类（非时限性），请使用[docs-inventory.zh-CN.md](../docs-inventory.zh-CN.md)。
